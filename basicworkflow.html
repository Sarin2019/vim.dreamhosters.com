Basic workflow in vim

The following is a very basic workflow in vim.

  1. Open the file: 
    [server]$ vim filename
  2. Change to insert mode by clicking the letter 'i'.
  3. Edit the file.
  4. Click 'ESC' to switch to COMMAND mode.
  5. Save changes one of two ways.
  6. If you want to exit without saving, click 'ESC' to enter COMMAND mode. Then type :q!. 
    [server]$ :q!
