How to change specific colors

You can change specific colors in your theme by using the highlight command.

The general syntax is highlight Group OutputDevice fontkey = value

For example: 

highlight Comment ctermbg=DarkGray
Notice that the 'OutputDevice' (cterm) and 'fontkey'(bg) are combined together.

Terminology

When running highlight you'll need to specify a few things. Below are a few explanations of the shorthand code used:

Groups

These are the most common output groups used:

Normal Refers to all normal text
Comment Comments within a program
Constant Program constants, such as numbers, quoted strings, or true/false
Identifier Variable identifier names. HTML tag color.
Title HTML header text color
Statement A programming language statement, usually a keyword like "if" or "while"
PreProc A preprocessor, such as "#include" in C
Type A variable datatype, such as "int"
Special A special symbol, usually used for special characters like "\n" in strings
Underlined Text that should be underlined
Error Text which contains a programming language error

Output Device

There are three types of output devices. The most common two are shown below with their optional values.

Output Device Explanation Possible values
cterm Color terminal bold, underline, reverse, italic, none
gui GUI terminal bold, underline, reverse, italic, none

There's also an option (term) for black and white only terminals.

Examples

Change the background and foreground color of all comments

This changes the background and foreground color of all comments in the entire document:

:highlight Comment ctermbg=DarkGray ctermfg=Black
:syntax enable

The :syntax enable command may be needed to get the colors to show.

Change the foreground color of text

This changes the normal text color in your document:

highlight Normal ctermfg=White guifg=#FFFFFF
Notice how you can add both cterm and gui statements in the same line.

Adding your customizations to the .vimrc file

You can add your changes directly into a .vimrc file the same way you typed them. This will allow the settings to load every time you open a file.

  1. Navigate to your user's home directory: 
    [server]$ cd $HOME
  2. Create a file named .vimrc 
    [server]$ touch .vimrc
  3. In that file add you customizations. For example: 
    color delek
syntax on

highlight Comment ctermfg=Green
highlight Normal ctermfg=White guifg=#FFFFFF
