Overview

vim offers several tools to format the file you're working on. One example is that you can set it to only allow lines to be a certain amount of characters.

Setting each line to allow a specific amount of characters

Let's say you need all lines in your file to never exceed 80 characters. This example shows how to set the word-wrapping variable.

  1. Open the file in vim.
  2. While still in COMMAND mode, enter the following: 
    [server]$ :set tw=80
  3. Click 'ENTER'.

When you edit the file now, all lines will cap at 80 characters.
