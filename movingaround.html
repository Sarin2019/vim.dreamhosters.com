Overview

vim offers many options to navigate through a file. This allows you to easily jump to different section.

Small movements (up, down, left, right)

The following table shows how to move one line up or down or one space right of left in COMMAND mode.

Move up one line: k
Move left one space: h Move right one space: l
Move down one line: j

Jumping between words

To jump forward a single word, click the letter: w

To jump back a word, click the letter: b

Moving to beginning or end of line

In COMMAND mode

To jump to the beginning of line, click the zero key: 0

To jump to the end of the line, click the dollar sign: $

In INSERT mode

Move to beginning of line by first clicking CTRL + O followed by Shift + I.

Move to end of line by first clicking CTRL + O followed by Shift + A.

Larger movements up and down

The following shows how to quickly scroll or jump up and down in a file in COMMAND mode.

Moving down a file

  • Scroll down page one line at a time: CTRL + e
  • Scroll half way down page: CTRL + d
  • Scroll fully down one page: CTRL + f

Moving up a file

  • Scroll up page one line at a time: CTRL + y
  • Scroll half way up page: CTRL + u
  • Scroll up full page: CTRL + b
