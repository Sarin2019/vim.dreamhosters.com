How to open a file

You can open a file in vim by running the following command: 

[server]$ vim filename

Saving your changes

Below are two ways to save your file changes.

Option #1

  1. After you've finished editing in INSERT mode, click the 'ESC' key to switch to COMMAND mode.
  2. Enter two capital letter Zs. For example. 
    [server]$ ZZ

Option #2

  1. After you've finished editing in INSERT mode, click the 'ESC' key to switch to COMMAND mode.
  2. Enter the following: 
    [server]$ :wq
  3. Click the 'ENTER' key to close out and save.

Close the file without saving

To close the file without saving any changes click 'ESC' to switch to COMMAND mode.

Enter the following:

[server]$ :q!
