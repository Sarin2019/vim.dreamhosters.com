Overview

Sometimes you may be working on a file that is hundreds or possibly thousands of lines long.

It's not practical to scroll down a file to a specific line. In vim, you can open the file at that exact line number and begin working.

Opening a file at a specific line number

  1. First, cut 'cat' to find which line in the file you need to open. For example, let's say you're looking for the word 'California'. Run the following to view which line of the file it is on: 
    [server]$ cat -n filename | grep -i "California"
          1237 California became a state in the year 1850
  2. From the output you can see you need to jump to line #1237. Open the file in vim like this: 
    [server]$ vim +1237 filename
  3. If necessary, you can view the line numbers in the file by running this in vim: 
    [server]$ :set nu!

You can now edit the file in vim and save changes from this specific line number.
