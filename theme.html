How to change your theme

  1. First, check your system for a list of themes already installed. If you're using OS X or Linux, it's most likely in the following directory: 
    [server]$ /usr/share/vim/vim73/colors
    NOTE: The /vim73 directory could be a different number such as vim72.

  3. Make a note of all the themes.
  4. Open your file in vim.
  5. Run the following to view what each theme looks like: 
    [server]$ :colorscheme delek
  6. Just change the theme name each time.
  7. Navigate to your user's home directory: 
    [server]$ cd $HOME
  8. Create a file named .vimrc.
  9. In that file add the following line, just change your themename: 
    color delek
syntax on
  10. When you now open a file in vim, the theme is automatically applied.
